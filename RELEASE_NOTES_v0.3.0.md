# AI Mobile v0.3.0 - delivery-first token-efficient orchestration

AI Mobile now keeps the current Codex task productive and delegates only genuinely independent bounded work.

## Changes

- Replaced the default project-manager loop with six compact delivery tools.
- Reduced always-loaded skill and MCP schema context by about 68%.
- Added one-pass resource inventory, immediate background dispatch, compact job readback, and deterministic verification.
- Kept Codex on the critical path with a configurable 15% shared-capacity reserve.
- Prevented Antigravity CLI, desktop, OAuth, and permission prompts without explicit lane authorization.
- Removed the separate raw DevTools MCP process; direct CDP UI access remains available on demand.
- Preserved advanced project-manager, provider, recovery, and durable-job capabilities behind an explicit compatibility flag.
- Added verification-cost controls: no premium-on-premium reassurance reviews and at most one justified failover.
