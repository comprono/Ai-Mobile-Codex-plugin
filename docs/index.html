AI Mobile Codex Plugin

Coordinate local AI resources across Codex projects

A community Windows plugin that inventories capacity once, keeps Codex on the highest-value critical path, coordinates independent projects, and returns compact project-specific evidence without polling loops.

View on GitHub Install instructions

What the plugin does

AI Mobile treats local AI platforms as changing resources. The visible Codex task is a lightweight project console with no project-file ownership. A deterministic coordinator assigns acceptance-linked work to separate Codex CLI, Claude Code, Antigravity, or optional Cursor workers under machine-wide provider, quota, RAM, file-ownership, and storage limits. Verified isolated patches are integrated once without a model review. No Goal, heartbeat, schedule, manager cycle, hidden continuation, polling feed, or desktop popup is created.

Who it is for

This community plugin is for OpenAI Codex users who have local Antigravity, Claude Code, Codex CLI, or headless Cursor resources and want them to work toward independently verified outcomes. It is designed for Codex tasks steered from desktop or mobile while execution remains on the linked Windows PC.

How it connects Codex to Antigravity 2.0

  • MCP tools: expose nine finite actions for task or portfolio start, bounded dispatch, collection, evidence, completion, cancellation, passive inventory, and private preferences.
  • Chromium DevTools: remains an on-demand advanced path for visible Antigravity UI state; it is not loaded automatically when Codex starts.
  • PowerShell helper commands: provide CLI-first Antigravity and Claude worker execution without hardcoding another user's machine paths.
  • Local Windows bridge: stores state outside projects, isolates editing workers in bounded Git worktrees, and does not bypass authentication, quota, billing, or safety controls.

Get started

Clone the plugin into your Codex plugins directory, then refresh or install it from your Codex personal marketplace.

git clone https://github.com/comprono/Ai-Mobile-Codex-plugin.git "%USERPROFILE%\plugins\ai-mobile"

Run a setup check from PowerShell:

powershell -ExecutionPolicy Bypass -File "%USERPROFILE%\plugins\ai-mobile\scripts\antigravity.ps1" setup

This is a community plugin, not an official Google or Antigravity product. See the GitHub repository for full technical instructions, security guidance, and release notes.
