AI Mobile Codex Plugin

Run Codex, Antigravity, and Claude as one project team

A community local Windows plugin that turns one Codex chat into a capacity-aware project manager: it discovers current models and limits, packages bounded context, coordinates CLI-first workers, handles failures, and integrates verified outcomes.

What the plugin does

AI Mobile treats local AI platforms as teams and their models as players. The current Codex chat remains a manager-only control room while separate native Codex agents, Claude Code, Antigravity, and optional Cursor workers execute bounded tasks in parallel when dependencies and ownership allow it. A protected Codex reserve retains management capacity; durable CLI jobs can continue through a zero-model-token supervisor and resume after a Codex reset. Native work is reserved before spawn and bound to the returned agent id, while pairwise-disjoint writer boundaries permit safe concurrency and overlaps remain serialized. The plugin builds an alias-safe dependency graph, selects models and effort levels from current capacity evidence, creates a transcript-free context capsule, and preserves durable continuity. Desktop UI is reserved for visible state that a CLI cannot represent.

Who it is for

This community plugin is for OpenAI Codex users who have local Antigravity or Claude Code resources and want them to work toward one shared project outcome. It also supports workflows started from ChatGPT or Codex on mobile and continued through Codex on the linked Windows PC.

How it connects Codex to Antigravity 2.0

  • MCP tools: expose project-manager plans, context capsules, passive resource inventory, Codex and provider capacity evidence, durable job state, and privacy checks.
  • Chromium DevTools: connects to the DevTools endpoint exposed by the Antigravity desktop app for visible UI inspection and handoff.
  • PowerShell helper commands: provide CLI-first Antigravity and Claude worker execution without hardcoding another user's machine paths.
  • Local Windows bridge: keeps the integration on the user's own computer and does not bypass Antigravity authentication, quota, billing, or safety controls.

Get started

Clone the plugin into your Codex plugins directory, then refresh or install it from your Codex personal marketplace.

git clone https://github.com/comprono/Ai-Mobile-Codex-plugin.git "%USERPROFILE%\plugins\ai-mobile"

Run a setup check from PowerShell:

powershell -ExecutionPolicy Bypass -File "%USERPROFILE%\plugins\ai-mobile\scripts\antigravity.ps1" setup

This is a community plugin, not an official Google or Antigravity product. See the GitHub repository for full technical instructions, security guidance, and release notes.
