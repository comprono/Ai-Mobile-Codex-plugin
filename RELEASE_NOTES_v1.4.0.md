# AI Mobile 1.4.0 - Director-CFO orchestration

AI Mobile 1.4.0 changes complex-project orchestration from one narrow worker cycle into a durable Director-CFO program.

- Tasks that fit within one minute bypass orchestration.
- Complex projects build authorized context before strategy.
- A strong planner creates the master plan, milestones, risks, team design, timeline, and acceptance path.
- The CFO forecasts the whole plan, protects reserves, and allocates models, permissions, concurrency, and time from live resources.
- Typed workers execute bounded packages while the visible Codex task remains a lightweight reporting console.
- Semantic failures receive strong reconciliation and a materially changed recovery plan before retry.
- Completion requires accepted outcome evidence, not worker activity.
- Existing canonical tasks can migrate in place without losing accepted evidence or creating another task.
- Installation fails closed unless Codex and Claude caches exactly match the complete source runtime.
- Restart verification requires the exact 1.4.0 version and SHA-256 build fingerprint before Spark resumes one Director campaign.

The legacy task tools remain available for existing durable tasks and diagnosis. New complex work starts with start-program, advances with run-program-campaign, and reports through program-report.
